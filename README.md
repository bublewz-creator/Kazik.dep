# NEONDROP — CS2 Cases & Upgrade (демо)

Стильное казино-демо в духе DinoDrop / CaseBattle: открытие кейсов с рулеточной
анимацией, апгрейд скинов, контракты обмена (trade-up) и инвентарь. Чистый
HTML/CSS/JS — **без сборки и зависимостей**.

> ⚠️ Это развлекательный/учебный проект. Используется только виртуальная валюта.
> Реальных ставок, платежей и вывода средств нет.

## Возможности

- **Кейсы** — 6 тематических кейсов с разными ценами и пулами скинов.
  Анимация рулетки (как в CS2), множитель открытия x1/x2/x3/x5, режим «Быстро».
  Шансы редкостей по реальной модели CS2 (Mil-Spec 79.92% … ★ 0.26%).
- **Апгрейд** — выбери свои предметы и цель; шанс считается честно как
  `стоимость / цель × 0.92`, колесо с иголкой крутится и определяет результат.
- **Контракт обмена (trade-up)** — закинь 3 скина, получи один на ступень выше.
- **Инвентарь** — продажа по одному или «продать всё», сортировки.
- **Баланс** в `localStorage`, бонус раз в 8 часов, демо-пополнение.
- **Лента живых дропов**, конфетти, звуки (WebAudio, без файлов), адаптив.
- Реальные скины и картинки CS2 подтягиваются из публичного
  [CSGO-API (ByMykel)](https://github.com/ByMykel/CSGO-API). Если интернета нет —
  работает офлайн-фолбэк с генерируемыми превью.

## Запуск

Самый простой способ — открыть `index.html` двойным кликом.
Для корректной работы загрузки скинов лучше запустить локальный сервер.

### Windows (Python уже есть)

Дважды кликни `start.bat` — он поднимет сервер и откроет браузер.

Или вручную:

```powershell
py -m http.server 8000
```

Затем открой http://localhost:8000

## Структура

Все файлы лежат в одной папке (плоско) — так проще заливать на GitHub Pages:

```
index.html     — разметка и подключение скриптов
styles.css     — тема, анимации, адаптив
icons.js       — встроенные SVG-иконки
data.js        — редкости, кейсы, шансы, расчёт цен, фолбэк-датасет
state.js       — баланс/инвентарь/статистика (localStorage)
skins.js       — загрузка скинов из API, пулы, алгоритм выпадения
effects.js     — конфетти, тосты, звуки, общий рендер карточек
cases.js       — кейсы и рулеточная анимация
upgrade.js     — апгрейд + колесо шанса
contract.js    — контракт обмена
inventory.js   — инвентарь и продажа
ui.js          — навигация, шапка, тикер дропов
app.js         — инициализация
```

> Важно: держи все файлы в корне репозитория (рядом с `index.html`),
> без подпапок `css/` и `js/`. Иначе пути к стилям и скриптам не совпадут.

## Алгоритмы (provably-fair-стиль)

- Цена скина детерминирована хэшем имени → стабильна между сессиями.
- Редкость кейса выбирается взвешенным рандомом по таблице `CASE_ODDS`.
- Износ (wear) добавляет ценовую вариативность.
- Шанс апгрейда ограничен диапазоном 2%…92%, заложен house edge 0.92.
